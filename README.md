# ЩО? ДЕ? КОЛИ? — База запитань

Локальний сайт для зберігання та перегляду запитань із зіграних пакетів.

## Структура файлів

```
qdkk-site/
├── index.html       ← Головна сторінка (список пакетів)
├── questions.html   ← Сторінка з запитаннями пакету
└── styles.css       ← Всі стилі
```

## Як запустити

Просто відкрийте `index.html` у браузері. Всі дані зберігаються в `localStorage` браузера.

> ⚠️ **Увага**: localStorage прив'язаний до браузера. Якщо хочете переносити дані між пристроями — скористайтесь функцією Експорту/Імпорту JSON (можна додати пізніше).

## Як додавати запитання

### Спосіб 1 — PDF імпорт (рекомендовано)
1. Натисніть **📄 Імпорт PDF**
2. Перетягніть або виберіть PDF файл (можна кілька)
3. Сайт автоматично витягне запитання

**Формат PDF, який добре розпізнається:**
```
Запитання 1. Текст першого запитання...
Відповідь: Правильна відповідь
Коментар: Додаткова інформація

Запитання 2. Текст другого запитання...
Відповідь: Відповідь 2
```

Або просто нумерований список:
```
1. Текст запитання
Відповідь: ...

2. Текст запитання
Відповідь: ...
```

### Спосіб 2 — Вручну
1. Натисніть **✏️ Вручну**
2. Заповніть форму
3. Додайте будь-яку кількість запитань

## Спойлери

На сторінці запитань відповіді приховані за спойлерами:
- **Натисніть** кнопку "Показати відповідь" для одного запитання
- Кнопка **"Показати всі відповіді"** — відкриває всі одразу
- Кнопка **"Сховати всі"** — закриває всі

## Якщо PDF не розпізнається добре

Відредагуйте JSON вручну або скористайтесь ручним введенням. Парсер шукає слова "Запитання", "Питання", "Question" або нумеровані списки (1., 2., тощо).
